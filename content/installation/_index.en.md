Karnak is distributed as a containerized application using Docker and Docker Compose. The [karnak-docker](https://github.com/OsiriX-Foundation/karnak-docker) repository contains all the necessary files to run Karnak with Docker Compose. Follow the steps described in the [readme](https://github.com/OsiriX-Foundation/karnak-docker#karnak-docker).

> [!INFO]
> Karnak is also available as a [portable distribution](../userguide/portable) for local use.
